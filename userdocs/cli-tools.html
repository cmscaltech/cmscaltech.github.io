CLI Tools

  • Kubectl install: See kubectl section here: https://kubernetes.io/docs/tasks/tools/ and install this on your local PC.
  • (Optional) Helm: Is the package manager for Kubernetes. Install link https://helm.sh/docs/helm/helm_install/>
  • (Optional) Krew: Plugin manager for kubeclt command-line tool. Install link https://krew.sigs.k8s.io/docs/user-guide/setup/install/
  • (Optional) If you plan to use Virtual machines (not containers) - you will need virtctl client installed. Install link https://kubevirt.io/user-guide/operations/virtctl_client_tool/
  • (Optional) K9s CLI - for easier to manage all your Kubernetes resources. Install link https://k9scli.io/topics/install/

How to get access to Tier2 Kubernetes?

  • Request a Tier2 account. Details are here: https://tier2.hep.caltech.edu/?page_id=11
  • Once account is created - ask Caltech admins hep-wheel@caltech.edu to create a namespace and kube configuration.

How to get access to Tier2 Kubernetes?

  • Request a Tier2 account. Details are here: https://tier2.hep.caltech.edu/?page_id=11
  • Provide project description. Details are here: https://github.com/cmscaltech/sdn-testbed
  • Once account is present and project description prepared - ask Caltech admins hep-wheel@caltech.edu to create a namespace and kube configuration.

How to get the kube config?

  • The Kube configuration file is stored in your home directory (login-[123].hep.caltech.edu). Once your Caltech Tier2/SDN account is active, you will get 2FA credentials and can ssh to any of the nodes.
  • CMS Caltech Tier2 Kube Config will be stored at: ~/.kube/config-caltech-tier2).
  • SDN Testbed Kube Config will be stored at: ~/.kube/config-caltech-sdn).
Be aware - the config token is valid only for 1 week and you will need to re-download it. Save the file as config and put the file in your PC ~/.kube folder.

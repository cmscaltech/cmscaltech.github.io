Working with Images

Pulling and Pushing Images

Ensure that you have docker installed on your machine
To pull and push Docker images from Harbor, you can use the 'docker' command-line tool

Pulling an image from Harbor

If the project that the image belongs to is private, you must sign in first:
docker login harbor.tier2-lb.ultralight.org
You can now pull an image:
docker pull harbor.tier2-lb.ultralight.org/repository/image-name:tag

Pushing an image to Harbor

docker tag local-image:tag harbor.tier2-lb.ultralight.org/repository/image-name:tag
docker push harbor.tier2-lb.ultralight.org/repository/image-name:tag
Note:
Tier2 Harbor_Address : harbor.tier2-lb.ultralight.org
Make sure to authenticate with Harbor if required.
You can use the docker login command to log in with your Harbor credentials before pushing or pulling images.
